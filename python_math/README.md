# Python Math

python provide some built-in methods and also provide a math module to perform mathematical tasks

# Built-in Math Functions

- min() - can be used to find lowest number in an interable object
- max() - can be used to find highest number in an interable object
- abs() - it returns the absolute(positive) value of a specified value
- pow(x,y)- to find power of a number(x)

### absolute: its a distance of a number from zero (always positive)

# The Math Module

- the most commonly methods are:
  - math.sqrt() -returns the square root of a number like give 64 in its parametr it will return 8
  - math.ceil() -will round upto the nearest integer (uper waly integer tak round krta h)
  - math.floor() - will round down to the naerest integer(neechy waly number tak round krta h )
  - math.pi - The math.pi constant, returns the value of PI (3.14...)

## Core Features Available

- Trigonometry: Functions like math.sin(), math.cos(), and math.tan().
- Power & Logarithms: Functions like math.log(), math.log10(), and math.pow(x,y)
- Rounding: Functions like math.ceil() (round up) and math.floor() (round down)
- Constants: Built-in definitions for math.pi, math.e, and math.tau

## round(number,ndigits)

- number - number we want to round
- ndigits - how many decimal places to keep
- If we don't specify the second argument, it rounds to the nearest whole number.
- round(3.7) → 4
